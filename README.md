- 원주교구 통합코스 마커 정보에서 영문 표기가 나오지 않도록 정리

- 기존 요안루갈다길과 188개 성지 공통 마커 유지


## v71 원주교구 경로 정리

- 원주교구 코스는 출발·도착과 경로 2km 이내 성지만 표시
- 성지 마커 팝업은 성지 이름만 표시
- 1-3 및 2-2 GPX 내부의 비정상 장거리 직선 연결을 트랙 세그먼트로 분리
- 전체코스 GPX를 분리된 세그먼트 기준으로 재생성
- 통합코스 번호 마커 생성 로직 제거, 출발·도착 마커만 유지

## v71 원주교구 통합 지도 중복 정리
- 님의 길 통합코스에서 정방향·역방향으로 겹치는 동일 경로선은 지도에 한 번만 표시합니다.
- 표시용 중복 제거와 내비게이션용 GPX 데이터는 분리하여, 선택한 코스의 실제 진행 방향은 그대로 유지합니다.
- 같은 성지가 이름 또는 근접 좌표로 중복될 때 지도에는 마커 하나만 표시합니다.
- 역방향 전환 시에도 GPX 세그먼트 경계가 유지되어 떨어진 구간이 직선으로 연결되지 않습니다.

## v74 수정
- 님의 길 2-2 최비르지타 순교길을 사용자가 첨부한 원본 GPX의 단일 연속 트랙(2,008개 좌표)으로 복원했습니다.
- 1길·2길·3길 전체코스와 님의 길 전체코스를 원본 세그먼트 기준으로 다시 생성했습니다.
- 님의 길 전체코스는 각 세부 코스별로 다른 색상을 사용합니다.
- 서울순례길 전체코스도 세부 순례길별 색상을 사용합니다.
- 한티가는길 전체코스는 원본 경로 세그먼트별 색상을 구분합니다.
- 겹치거나 반대 방향으로 이용하는 경로도 표시 단계에서 삭제하지 않고 각 색상으로 그리며, 내비게이션 방향 데이터는 원본 그대로 유지합니다.

## v75 변경사항
- 한티가는길 전체코스를 1~5코스별 고대비 색상으로 표시
- 서울순례길 전체코스를 1~3코스별 고대비 색상으로 표시
- 님의 길 전체코스 색상 팔레트를 인접 구간끼리 확실히 구분되도록 변경
- 겹치는 경로는 선 굵기를 달리해 여러 색이 함께 보이도록 처리
- 지도 상단에 선택한 코스의 총거리(km) 배지 표시
