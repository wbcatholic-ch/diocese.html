# 순례길 내비게이션 PWA V36

- 기준 파일: V35 전체 ZIP.
- 전국 순례길 목록과 카드형 상세 코스 디자인을 유지했습니다.
- 목록 카드를 누르면 외부 사이트로 바로 이동하지 않고, 현재 독립 PWA에 연결된 상세 코스 화면으로 이동합니다.
