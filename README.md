# 순례길 내비게이션 PWA V41

- 기준 파일: V40 전체 ZIP.
- 전국 순례길 목록과 카드형 상세 코스 디자인을 유지했습니다.
- 목록 카드를 누르면 외부 사이트로 바로 이동하지 않고, 현재 독립 PWA에 연결된 상세 코스 화면으로 이동합니다.
- 외부 공식 사이트는 카드 하단의 `웹사이트 열기` 또는 상세 화면의 `공식 홈페이지 / 상세보기` 버튼으로만 열립니다.

## v40 수정 사항

- 한티가는길 1~5코스 카드 오른쪽에 `코스소개` 정사각형 버튼을 추가했습니다.

- 한티가는길 코스 소개 URL은 `routes/hanti-route-data-v1.js`의 각 코스 데이터 owner에 저장했습니다.
- 서울순례길 코스 소개 URL은 `routes/seoul-pilgrimage-routes.js`의 각 코스 데이터 owner에 저장했습니다.
- 중복 리스너나 하단 override 없이 `js/route-nav.js`의 상세 카드 렌더링 owner와 `css/route-nav.css`의 코스 카드 스타일 owner를 직접 수정했습니다.
