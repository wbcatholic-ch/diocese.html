# 순례길 내비게이션 독립 PWA

가톨릭길동무 본앱에 나중에 병합하기 위해 분리한 순례길 내비게이션 전용 PWA입니다.

## 지도 기준

- 본앱의 다른 지도 카테고리와 동일하게 **Kakao Map**을 사용합니다.
- OpenStreetMap/Leaflet 지도는 제거했습니다.
- 성지·성당·피정 지도, 일반 길찾기, 카카오내비 경로검색 기능은 넣지 않았습니다.
- 순례길 GPX 경로선, 스탬프 지점, 내 위치, 따라가기만 유지합니다.

## 포함된 순례길

- 한티가는길
- 서울순례길 1코스 · 말씀의 길: 실제 GPX
- 서울순례길 2코스 · 생명의 길: 실제 GPX
- 서울순례길 3코스 · 일치의 길: 실제 GPX
- 김대건 신부 치명 순교길: 실제 GPX
  - 서울순례길 4코스가 아니라 별도 순례길로 분리

## GitHub Pages 테스트 전 확인

Kakao Map을 사용합니다. 도메인 등록이 이미 되어 있는데도 지도가 멈추면, 예전 서비스워커/사이트 데이터 캐시가 남아 이전 파일이 실행되는지 먼저 확인해야 합니다.

예: `https://bong0219.github.io`

## 이번 수정

- OSM/Leaflet 사용 제거
- Kakao Map 기준 복원
- Chrome 안내 배너 조건 유지
- 초기 진입 시 반드시 순례길 목록이 먼저 보이도록 hidden 우선순위 유지
- Kakao Map SDK 로딩 타임아웃을 SDK 파일 로딩뿐 아니라 `kakao.maps.load()` 완료까지 적용
- 지도 컨테이너를 `map-layer`와 로딩 오버레이로 분리
- 김대건 신부 치명 순교길을 서울순례길 4코스가 아닌 별도 순례길로 정정
- PWA 캐시 버전 업데이트: `pilgrimage-route-nav-v10-kakao-stable-20260707`
