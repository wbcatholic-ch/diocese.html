# 순례길 내비게이션 독립 PWA

가톨릭길동무 본앱에 나중에 병합하기 위해 분리한 순례길 내비게이션 전용 PWA입니다.

## 수정 원칙

- 패치식 덧붙임 수정 금지
- 같은 기능을 여러 곳에 분산 금지
- 중복 코드 생성 금지
- 잘못 수정된 코드는 남기지 않고 원래 잘못된 위치에서 직접 정리
- 지도, 따라가기, 방향, 임시 상태, 종료 처리는 각각 하나의 주인 로직에서 관리

## 지도 기준

- 본앱의 다른 지도 카테고리와 동일하게 **Kakao Map**을 사용합니다.
- OpenStreetMap/Leaflet 지도는 사용하지 않습니다.
- 성지·성당·피정 지도, 일반 길찾기, 카카오내비 경로검색 기능은 넣지 않았습니다.
- 순례길 GPX 경로선, 스탬프 지점, 내 위치, 따라가기만 유지합니다.

## 첫 화면 구조

첫 화면은 `전국 가톨릭 순례길` 목록입니다.

안내 문구: `목록을 누르면 상세 코스로 이동합니다.`

첫 목록에는 아래 8개만 표시합니다.

1. 한티가는길 — 대구대교구
2. 천주교 서울 순례길 — 서울대교구
3. 성지순례길 ‘디딤길’ — 수원교구
4. 원주교구 순례길 ‘님의 길’ — 원주교구
5. 천주교 순례길 — 전주교구
6. 신부님과 함께 걷는 순례길 — 안동교구
7. 순례길 — 광주대교구
8. Holy Journey SANTO VIAGGIO — 제주교구

목록을 누르면 외부 홈페이지로 바로 이동하지 않고, 상세 코스 카드로 들어갑니다. 공식 홈페이지/상세보기는 상세 카드 하단 버튼으로만 엽니다.

## 현재 내비게이션 연결 상태

- 한티가는길 전체 및 1~5코스
- 천주교 서울 순례길 1~3코스 전체
- 천주교 서울 순례길 1코스 말씀의 길
- 천주교 서울 순례길 2코스 생명의 길
- 천주교 서울 순례길 3코스 일치의 길
- 김대건 신부 치명 순교길
- 원주교구 순례길 `님의 길` 1-1 큰 첨례길
- 원주교구 순례길 `님의 길` 1-2 네 공소길

도원동/회사 근처 테스트 GPX 데이터는 프로젝트 내부에는 남아 있지만, 첫 화면 전국 목록에는 표시하지 않습니다.

## 천주교 서울 순례길 상세 정보

상세 코스 카드에 아래 정보를 표시합니다.

- 1~3코스 총 44.1km
- 1코스 8.7km · 3시간 40분
- 2코스 5.9km · 2시간 30분
- 3코스 29.5km · 8시간
- 김대건 신부 치명 순교길 12.7km · 4~5시간

서울순례길 전체에는 김대건 신부 치명 순교길을 포함하지 않고, 별도 버튼으로 유지합니다.

## 원주교구 순례길 `님의 길`

상위 그룹: `1길 최양업 신부님의 길`

연결 코스:

- 1-1 큰 첨례길: 풍수원성당 ~ 창촌공소, 10.3km · 3시간
- 1-2 네 공소길: 창촌공소 ~ 옛 구재공소, 15.4km · 4시간

업로드된 GPX 원본 저장 경로:

- `gpx/nimui-gil-1-1-big-cheomnye.gpx`
- `gpx/nimui-gil-1-2-ne-gongso.gpx`

내비게이션용 points는 `routes/nimui-gil-routes.js` 데이터 owner에서 직접 연결했습니다.

## 따라가기 기준

- 경로선 위 진행 방향 화살촉 표시
- 정방향/역방향 전환은 시작 전만 가능
- 지나온 길과 앞으로 갈 길 분리 표시
- 지도 조작 중에는 `지도 확인 중 · 15초 후 내 위치로 복귀`
- 내위치 버튼은 즉시 현재 위치로 복귀
- 종료 시 `지금까지 이동한 경로가 삭제됩니다.` 확인 후 임시 이동 경로 삭제
- 8시간 미만 복귀는 따라가기 상태 복원
- 8시간 이상 미사용 시 임시 따라가기 상태 자동 초기화
- 완료 기록 저장 기능 없음, 복귀용 임시 내비게이션 상태만 저장
