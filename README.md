# 순례길 내비게이션 PWA v81

- 원주교구 님의 길 1-5·3-1 코스명 라벨을 전체 경로의 실제 거리 중앙으로 이동
- 여러 GPX 세그먼트로 나뉜 동일 코스는 세그먼트를 합산해 중앙점 계산
- 한티가는길 번호 마커를 기존보다 낮은 줌부터 표시
- 전체 코스명 라벨을 더 낮은 줌에서도 작게 표시하고, 지나치게 축소된 경우만 숨김
- 공통 줌별 라벨 크기 유지


## v83
- 한티가는길 성지 마커는 성지 Excel 자료에서 생성된 공통 성지 좌표를 사용합니다.

- 경로 스탬프 좌표를 성지 마커 좌표로 사용하던 로직을 제거했습니다.
